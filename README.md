# ELEMENT TYPING — 元素記号タイピング

元素記号を楽しく覚えるタイピング学習ゲーム。ブラウザだけで動きます(インストール不要)。

## 🎮 遊び方

**▶ プレイはこちら → https://mi-po913.github.io/element-typing/**

落ちてくる元素の答えを打ち込んで **Enter で確定**して撃ち落とそう。

- 打ち間違いは **Backspace で修正OK**(タイプミスは失敗になりません)
- 間違った答えで確定するとヒントが表示されます
- 下のラインに着くとライフが減ります
- ⌨️ IMEはOFF(半角英数)でプレイしてください / ESCで中断

## モード

| モード | 内容 | 例 |
|---|---|---|
| 🧪 記号モード | 元素名を見て記号をタイプ | 水素 → `h` |
| 🔤 読みモード | 記号を見て読みをローマ字でタイプ | H → `suiso` |
| 📜 語呂合わせモード | 「水兵リーベ僕の船…」をタイプ | 水兵 → `suihei` |

## レベル・出題順

- 🏫 **中学レベル**: 原子番号 1〜20(H〜Ca)
- 🎓 **高校レベル**: 原子番号 1〜36(H〜Kr)+ Ag・Sn・I・Ba・Pt・Au・Hg・Pb・U
- 📋 周期表順 / 🎲 ランダム を選択可能

## 技術

HTML / CSS / JavaScript のみの単一ファイル構成。ローマ字入力は `shi/si`、`fu/hu`、促音、`n/nn` などの表記ゆれに対応。
